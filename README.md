# 🚗 智能客户管理系统 - 理想汽车版

基于CELN模型和探需九问的智能客户画像分析系统，专为汽车销售场景打造。

## 🌟 核心功能

### 专家视图
- **智能客户分析**：输入自然语言描述，自动生成客户画像
- **CELN模型评估**：客户意向(C)、能源类型(E)、品牌偏好(L)、购买时机(N)
- **探需九问分析**：9个维度全面评估客户需求探查完成度
- **智能话术推荐**：根据跟进次数和客户状态自动生成1-2条精准话术
- **一键复制话术**：点击即可复制，快速应用到实际沟通
- **历史记录管理**：自动保存最近10条分析记录

### 店长视图
- **Excel批量导入**：支持团队客户数据批量分析
- **团队统计看板**：实时展示商机分布、探需完成度等关键指标
- **专家探需雷达图**：可视化展示每个专家在9个维度的探需能力
- **薄弱项分析**：自动识别团队和个人探需薄弱点，提供改进话术
- **快速搜索筛选**：按姓名、商机等级、探需完成度快速定位客户
- **数据持久化**：刷新页面数据不丢失

## 🎯 技术特点

- ✅ **纯前端实现**：无需服务器，双击即可运行
- ✅ **离线可用**：无网络环境也能正常使用
- ✅ **智能识别**：支持车型简写（tsl→特斯拉）、日期提取、上下文分析
- ✅ **数据安全**：所有数据存储在本地浏览器
- ✅ **零学习成本**：界面友好，操作直观

## 📖 使用指南

### 专家日常工作流
1. 打开系统，选择"专家视图"
2. 输入客户信息（自然语言描述即可）
3. 点击"智能分析客户画像"
4. 查看分析结果和话术推荐
5. 点击"复制"按钮，粘贴到企业微信使用

### 店长数据分析流程
1. 切换到"店长视图"
2. 上传团队客户Excel表格
3. 查看整体统计数据
4. 点击专家名称查看详细分析
5. 查看雷达图识别薄弱维度
6. 根据改进建议进行针对性培训

## 💡 核心算法

### CELN模型
- **C (Customer Intention)**: 购车意向评估
- **E (Energy Type)**: 能源类型偏好
- **L (Brand Loyalty)**: 品牌偏好度
- **N (Timing Necessity)**: 购买时机判断

### 探需九问
1. 能源形式-购车意向
2. 用车情况（置换/增购）
3. 用车场景
4. 购车预算
5. 家庭结构
6. 决策人
7. 关注功能
8. 竞品对比
9. 购买时间

## ⚠️ 注意事项

1. **浏览器兼容性**：建议使用Chrome、Edge或Safari最新版本
2. **数据隐私**：所有数据仅存储在本地浏览器，不会上传到服务器
3. **清理缓存**：清理浏览器缓存会导致历史记录丢失
4. **Excel格式**：导入Excel需包含字段：专家姓名、客户姓名、商机等级、客户描述
